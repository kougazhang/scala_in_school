## 基础知识

### List: 链表

```scala
// 普通方式定义
scala> List(1,2,3)
res9: List[Int] = List(1, 2, 3)
// 函数式方法定义
scala> 1 :: 2 :: 3 :: 4 :: Nil
res10: List[Int] = List(1, 2, 3, 4)

```

### Set: 集

集没有重复, 无序

```scala
>Set(1,2,2)
Set(1,2)
```

### Seq: 序列

序列有一个给定的顺序. Seq 是一个特质. List 是序列的实现. Seq 也是一个工厂单例对象, 可以用来创建列表.

```scala
scala> Seq(1,1,2)
// 返回值是 List, 说明是工厂单例对象.
res11: Seq[Int] = List(1, 1, 2)
```

### 映射: Map

Map 是键值容器.

```scala
Map('a'->1, 'b'->2)
```
